
# 问题解答

## 问题一：FutureWarning: Passing (type, 1) or '1type' ...

* 问题描述

```
FutureWarning: Passing (type, 1) or '1type' as a synonym of type is deprecated; in a future ver...
```

* 问题解析

`numpy`版本过高

* 解决方案

```
pip install numpy==1.16.4
```
